# DevTrack – Job Application Tracker

A full-stack web application designed to help job seekers organize, track, and manage their job applications in one centralized platform. DevTrack simplifies the job search process by keeping application details, progress, recruiter notes, and follow-up reminders organized and accessible.

**Project Status:** In Development

---

## Overview

DevTrack is a personal full-stack project built to solve a real-world problem encountered during the job hunting process. It serves as both a daily productivity tool and a learning project for modern web development.

The application enables users to securely manage their job applications, monitor each application's progress, and keep important information such as recruiter contacts, interview notes, and follow-up schedules in one place.

---

## Features

- Secure user authentication using JWT
- Individual user accounts with private application data
- Create, update, and delete job applications
- Track application progress through multiple hiring stages
- Manage follow-up reminders
- Store recruiter information and interview notes
- Responsive interface for desktop and mobile browsers
- Organized dashboard for viewing and managing applications

## Development Approach

Rather than following tutorials line by line, this project follows a learn-by-building approach. Each feature is implemented incrementally with an emphasis on understanding the underlying concepts before writing code.

The objective is not only to complete the application but also to develop stronger software engineering skills by building a real-world project from the ground up.
